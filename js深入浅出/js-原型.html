    # Object.prototype === object.__proto__

    1. 每个函数都有一个 prototype 属性，指向它实例的原型

    2. 每个对象都有一个 _proto_ 属性，指向它的原型

    # Function.prototype === Function.__proto__

    • Function 本身是个函数，而所有函数都是 Function 的实例，因此Function._proto_ === Function.prototype

    # Object__proto__ === Function.prototype

    • Object 是一个函数，而所有函数都是 Function 的实例，因此 Object._proto_ === Function.prototype
